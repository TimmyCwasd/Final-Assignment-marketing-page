Prototype code

The code that we have installed on our microbit is responsible for the LED temperature display as well as the automatic factor of the Breasy system.
  1. The "forever" block of code that we encase everything else in ensures that the Microbit is always displaying the temperature and checking for temperature differences
  2. "show number temperature C" displays the temperature detected from the sensor on the LEDs of the Microbit. The value detected by the sensor is sent to the processor, which displays it on the LEDS.
  3. We use if statements to check when the temperature is less than or greater than 25 degrees celsius, which then sends a positive signal to the transistor, telling it to allow power to pass through.
